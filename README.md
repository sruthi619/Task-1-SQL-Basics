# Task 1: SQL Basics – Intern Training

## Objective
To understand SQL fundamentals by creating a database, table, inserting data, and retrieving records.

## Tools Used
- MySQL Community Server
- MySQL Workbench

## What Was Done
- Created a database named `intern_training_db`
- Created a `students` table
- Inserted 5 student records
- Retrieved data using SELECT queries

## Files Included
- `intern_training_db.sql` – Contains all SQL queries used in this task

## Outcome
Learned basic SQL syntax, database structure, and data retrieval.
